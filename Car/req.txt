Exotic Motors - Granular Features & Requirements Audit

============= AUTHENTICATION & ACCESS PROTOCOLS =============
1.  Client Registration: Create a new user identity in the system.
2.  Client Login: Authenticate using email and password credentials.
3.  Admin Login: Access the executive portal via a secure administrative gateway.
4.  Logout: Securely terminate the current session and clear tokens.
5.  Role-Based Access Control (RBAC): Restricted access to admin modules based on user roles (Admin, WorkshopManager).
6.  Session Integrity: Maintain persistent authentication across page reloads.

10. View Vehicle Details: Access deep specifications, high-res images, and pricing for a specific car.
